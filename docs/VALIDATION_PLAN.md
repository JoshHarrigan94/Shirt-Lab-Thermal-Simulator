# Validation Plan

## Cheap real-world test kit

- kitchen scale
- thermometer
- weather app
- HR watch/strap
- standard route
- cotton tee
- punched-hole tee
- polyester running tee
- mesh tee

## Protocol

1. Weigh dry shirt.
2. Run fixed route at similar effort.
3. Record temperature, humidity, wind and pace.
4. Weigh shirt immediately after.
5. Record comfort, cling, cooling and chill scores.
6. Record drying time indoors/outdoors.

## Initial validation metrics

- predicted wetness vs shirt weight gain
- predicted comfort vs subjective comfort
- predicted cooling advantage vs perceived cooling
- humidity penalty accuracy
- perforation benefit under wind/no-wind conditions
