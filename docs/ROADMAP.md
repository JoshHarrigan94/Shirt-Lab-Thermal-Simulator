# Roadmap

## Pass 1 — Static 3D product scaffold

- GitHub Pages-ready structure
- 3D torso/shirt view
- material, fit and perforation presets
- heuristic thermal model
- comparison mode

## Pass 2 — Better product geometry

- improve shirt silhouette
- add sleeves and hem options
- add front/back orientation controls
- add zone hover labels

## Pass 3 — Perforation designer

- click/paint holes onto garment zones
- open area calculator
- random/grid/gradient pattern generator
- structural weakness score

## Pass 4 — Experiment logger

- log real run conditions
- pre/post shirt weight
- perceived comfort
- weather import/manual entry
- compare prediction vs reality

## Pass 5 — Surface-cell simulation

- convert zone model into surface-cell model
- add heat/moisture transfer between neighbouring cells
- simulate air pocket behaviour

## Pass 6 — WebGPU/CFD-lite

- approximate airflow field
- simulate moving runner wind
- export visual simulation clips
